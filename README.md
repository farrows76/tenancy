# Tenancy
Simple middleware to set up a Multitenant environment.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'tenancy'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install tenancy
```
